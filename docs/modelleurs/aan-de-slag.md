# Aan de slag

## Installatie

Om modellen te maken in LinkML en draw.io heb je enkel [VS Code](https://code.visualstudio.com/) nodig met de [Draw.io Integration](https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio) extensie. De installatie vereist geen beheerrechten.

## Repository openen

De gemakkelijkste manier is om de repository remote te openen:

1. Klik op `Connect to...`
2. Kies `Open Remote Repository...`
3. Voer de URL in: `https://github.com/sefanja/nbnl-conceptuele-informatiemodellen`

## Modellen maken en bewerken

Je kunt nu bestaande modellen bewerken, zowel de LinkML- als de drawio-bestanden, gewoon door erop te klikken. Om een nieuw model te maken, kopieer je `/hulpmiddelen/voorbeeldregister/` naar `/modellen/`. In de bestanden vind je voorbeelden om snel aan de slag te gaan.

Om je wijzigingen op te slaan, klik je links op `Source Control` om deze te 'committen'.

We werken samen op de `main` branch. Bewerk daarom alleen je 'eigen' modellen om mergeconflicten te voorkomen.
