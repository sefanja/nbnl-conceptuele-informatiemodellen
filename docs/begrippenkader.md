# Begrippenkader

Het [begrippenkader](https://begrippen.netbeheernederland.nl) definieert de soorten dingen die worden beheerd door de Nederlanse [transmissie- en distributiesysteembeheerders](https://wetten.overheid.nl/BWBR0050714/2026-01-01) bij de uitvoering van de economische activiteiten [3513 (Transmissie van elektriciteit)](https://sbi-api.cbs.nl/api/SBIData/SBIInfo/3513), [3514 (Distributie van elektriciteit)](https://sbi-api.cbs.nl/api/SBIData/SBIInfo/3522), [4950 (Transport via pijpleidingen)](https://sbi-api.cbs.nl/api/SBIData/SBIInfo/4950) en [3522 (Distributie van gasvormige brandstoffen via leidingen)](https://sbi-api.cbs.nl/api/SBIData/SBIInfo/3522).
